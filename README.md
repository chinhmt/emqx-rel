emqx-rel
========

The Release Project for EMQ X Broker.

NOTICE: Requires Erlang/OTP R21.0+ to build since EMQ X R3.2

# Build

```
git clone https://github.com/emqx/emqx-rel.git emqx-rel
cd emqx-rel && make
_build/cloud_dev/rel/emqx/bin/emqx console
```

# Test

```
make test-cloud_dev
```
